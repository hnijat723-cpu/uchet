# Учёт — клиенты и расходы

Личный инструмент: с одной стороны — клиенты и их платежи (доход), с другой —
твои личные расходы. Одна HTML-страница, без сервера и без интернета.

## Ссылка (телефон и компьютер)

**https://hnijat723-cpu.github.io/uchet/**

На телефоне интерфейс адаптирован под приложение: нижняя панель вкладок с
иконками, поля во всю ширину, отступы под Dynamic Island. На компьютере —
прежний вид с вкладками сверху.

### Установить на телефон как приложение

- **iPhone (Safari):** открой ссылку → кнопка «Поделиться» (квадрат со стрелкой) →
  «На экран Домой». Появится иконка «Учёт», открывается на весь экран, работает
  без интернета.
- **Android (Chrome):** открой ссылку → меню «⋮» → «Установить приложение»
  (или «Добавить на главный экран»).

Данные на телефоне и на компьютере — **раздельные** (у каждого свои). Телефон —
главное устройство. Чтобы перенести — «Экспорт» на одном, «Импорт» на другом.

**Удаление — в два нажатия:** первое нажатие «Удалить» превращает кнопку в
«Точно?», второе — удаляет. Сделано так намеренно: в приложении с домашнего
экрана iPhone обычное окно подтверждения не открывается.

## Как обновить приложение

Правишь файлы → `git commit` → `git push`. GitHub Pages обновится за минуту.
Если в браузере осталась старая версия — обнови страницу дважды (первый раз
service worker подтягивает новое, второй показывает). При заметных правках
подними версию кэша `CACHE` в `sw.js` — старый кэш очистится сам.

## Как открыть локально (без интернета)

Двойной клик по `index.html`. Но основной способ — ссылка выше.

## Две вкладки

**Клиенты (доход)**
- Клиент: имя, ежемесячная оплата, **себестоимость в месяц** (реклама, подрядчики,
  инструменты), заметка. Кнопки: изменить, пауза, удалить.
- Платежи — по каждому клиенту сумма и дата, видна вся история.
- Остаток за месяц — сколько клиент ещё должен внести до месячной суммы.
- **Прибыль с клиента** = ежемесячная оплата − себестоимость (бейдж на карточке).
- Сводка сверху: ожидается в месяц, себестоимость в месяц, **чистая прибыль в
  месяц** (доход − себестоимость), получено в этом месяце, остаток, число клиентов.
  Личные расходы сюда не входят — они в своём балансе на вкладке «Расходы».

**Приходы (разовые/хаотичные)**
- Для разовых заказов: имя клиента, **всего** (сколько должны отдать) и **взято**
  (сколько уже отдали). «Осталось» считается само; когда взято = всего — «закрыто».
- Правится прямо в карточке (без всплывающих окон): меняешь число — итог сразу.
- Сводка: должны взять, уже взяли, осталось получить.

**Сотрудники**
- Список людей по ролям (фотограф, видео, моушн-дизайн…). У каждого — имя, роль,
  заметка и история выплат (сумма, дата, за что).
- По каждому: выплачено в этом месяце, всего выплачено, число выплат.
- Сводка: выплачено сотрудникам в этом месяце, всего, число сотрудников.
- Пока это отдельный учёт — на «чистую прибыль» клиентов не влияет.

**Расходы**
- Записываешь: куда/на что, сумму, дату, комментарий. Список сгруппирован по месяцам.
- Сводка: потрачено в этом месяце, получено в этом месяце, **баланс месяца**
  (доход − расход; зелёный в плюс, красный в минус) и потрачено всего.

## Важно: где лежат данные

Данные хранятся **только в этом браузере на этом компьютере** (localStorage).
Их не видно на других устройствах, и они пропадут, если очистить данные браузера.

Поэтому регулярно делай **бэкап**: кнопка «↓ Экспорт» сохраняет файл `учёт-ДАТА.json`
(в нём и клиенты, и расходы). Кнопка «↑ Импорт» загружает его обратно.
